# SCN - Movie Web App

SCN is a modern movie discovery web application built with React and TMDB API.  
It allows users to browse trending movies, search for titles, explore details, and manage a personal watchlist.

## Features

- Trending movies showcase
- Search movies with debounce
- Discover movies by genre
- Movie details page
- Watchlist system
- Authentication UI
- Responsive design

## Tech Stack

- React
- Vite
- Tailwind CSS
- Bootstrap Icons
- Axios
- TanStack Query
- Redux Toolkit
- Swiper.js
- TMDB API
